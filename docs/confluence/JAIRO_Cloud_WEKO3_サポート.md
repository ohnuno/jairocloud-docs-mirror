# JAIRO Cloud（WEKO3）サポート

_Source: <https://nii-auth.atlassian.net/spaces/JAIROCloudWEKO3/overview>_

国立情報学研究所（以下「NII」）により運営されてきた「JAIRO Cloud」は、2016年7月オープンアクセスリポジトリ推進協会（以下「JPCOAR」）の設立に伴い、JPCOARとNIIとの共同運営となりました。ユーザ窓口等の運用はJPCOAR、開発はNIIが担います。2017年4月からJAIRO CloudはJPCOAR会員向けサービスとなりました。

**重要なお知らせ**

「JAIRO Cloud（WEKO3）サポート」を掲載しておりましたmeatwikiは2025年12月末に運用終了となりました。

**2025/11/28【重要】リリースノート等の掲載場所移行について**

meatwiki上に公開しておりました基本マニュアル以外のコンテンツ（リリースノート等）につきましては、本サイトで引き続き更新を行います。  
12月2日（火）12:00より、meatwikiのURLにアクセスした場合、本サイトへ自動転送され、meatwikiを参照することはできません。

詳細は、2025年11月11日付アナウンス「【重要】「JAIRO Cloud（WEKO3）サポート」移行に関するお知らせ（2025年11月）」をご確認ください。  
[事務局からのお知らせ](/wiki/spaces/JAIROCloudWEKO3/pages/43553760/JAIRO+Cloud)

**2025/10/21【重要】基本マニュアルの掲載場所移行について**  
meatwikiは2025年12月末に運用終了となります。  
[「JAIRO Cloud（WEKO3）基本マニュアル」](/wiki/spaces/JAIROCloudWEKO3/pages/43548700/JAIRO+Cloud+WEKO3)については、JPCOAR Webサイトに移行して引き続き公開します。

移行後の基本マニュアルは[こちら](https://jpcoar.org/support/jairo-cloud/manual/)

詳細は[JPCOAR Webサイトのお知らせ](https://jpcoar.org/news/3110/)をご確認ください。

**重要なお知らせ**

**2025/3/5【重要】メーリングリスト送信制限について**  
JAIRO Cloudの問い合わせ用メーリングリストへの一日あたりの投稿件数の上限について、下記のとおり一斉アナウンスを行っております。  
詳細をご確認いただき、本文中のエラーメッセージが届いた場合には、時間をおいて再度ご投稿ください。  
なお、本制限のため、MLへご投稿後、JAIRO Cloud事務局からの受領連絡およびクローズのみの連絡は控えさせていただいております。

[JAIRO Cloud事務局からのお知らせ](/wiki/spaces/JAIROCloudWEKO3/pages/43553760/JAIRO+Cloud)

**重要なお知らせ**

**2024/9/19【重要】著者DB一括登録およびアイテム一括削除機能の一時中止について**  
著者DB一括登録およびアイテム一括削除を行った際に、不具合が発生することが確認されています。  
そのため、著者DB一括登録およびアイテム一括削除は行わないでください。

（2026/3/19追記）v2.0.0以降、アイテム一括削除は行っていただけます。著者DB一括登録については引き続きお控えください。

```
2024/9/19付メールアナウンス
件名：（ログイン再開のご案内）【2024年9月9日（月）10:00～19日（木）12:00】JAIRO Cloudアップデート終了と注意事項
送信元：JAIRO Cloud事務局
送信先：JAIRO Cloud利用機関

※ 関係個所のみ抜粋

1）著者DB一括登録・アイテム一括削除機能の一時中止について
JPCOAR JAIRO Cloud Community MLのお問い合わせから、著者DBの一括登録及びアイテムの一括
削除機能について、影響の大きな不具合が発生することが確認されました。
調査・解消まで時間を要する見込みであり、回避策のご案内も難しいことから、
以下の機能を一時的に利用中止とさせていただきます。
利用可能となりましたら、改めてアナウンスいたします。

○対象機能
１．Administration > 著者DB管理 > 一括登録
　・Administration > 著者DB管理 > 一括登録 を用いたデータ更新が
　　既存のデータが更新されずに新たな著者として登録されます。
　・登録中に一括登録の進捗状況の取得に失敗し、サーバー内部エラーが発生する場合があります。
　ほか

　■参考：JAIRO Cloud（WEKO3）基本マニュアル > 著者DB管理
　　https://meatwiki.nii.ac.jp/confluence/pages/viewpage.action?pageId=63868529

２．Administration > アイテム管理 > 一括削除
　・アイテムの一括削除機能を行った際に、アイテムが削除されないまたは
　　意図しないアイテムが削除されます。

　■参考：JAIRO Cloud（WEKO3）基本マニュアル > アイテム一括登録（インポート） > 4. アイテムの一括削除
　　https://meatwiki.nii.ac.jp/confluence/pages/viewpage.action?pageId=63868511#id-アイテム一括登録（インポート）-4.アイテムの一括削除

本件への問い合わせはjc-office@nii.ac.jp までお願いします。
```
