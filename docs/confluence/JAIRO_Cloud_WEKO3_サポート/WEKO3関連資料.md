# WEKO3マニュアル
JAIRO Cloudを構成するシステム「WEKO3」について、v1.0.7以降のマニュアルをGitHubを用いて公開しています。

「[JAIRO Cloud（WEKO3）基本マニュアル](https://jpcoar.org/support/jairo-cloud/manual/)」は、以下のマニュアルをJAIRO Cloudユーザ向けにアレンジして作成したもののため、アップデートの際には、本ページのマニュアルが先に更新されます。閲覧タイミングによっては、「[JAIRO Cloud（WEKO3）基本マニュアル](https://jpcoar.org/support/jairo-cloud/manual/)」は最新のシステムの状況を反映していないことがありますので、最新情報は以下をご確認ください。  
なお、JAIRO Cloudでは操作・利用できない内容も含まれておりますが、ご了承ください。

- システム管理マニュアル（v2.0.0、v2.0.2）：管理画面［Administration］の操作マニュアル

  - <https://rcosdp.github.io/weko/v2.0.0/admin/>
- ユーザ操作マニュアル（v2.0.0、v2.0.2）：管理画面［Administration］以外の操作マニュアル

  - <https://rcosdp.github.io/weko/v2.0.0/user/>
