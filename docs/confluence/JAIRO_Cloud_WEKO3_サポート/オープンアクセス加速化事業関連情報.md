## ●JC移行支援キット
DSpace等のリポジトリシステムからJAIRO Cloud（WEKO3）へのシステム移行をご検討の皆さまを支援する情報をご提供いたします。

なお、JAIRO Cloudの利用には、JPCOAR（オープンアクセスリポジトリ推進協会）への入会とJAIRO Cloudの利用申請が必要となります。詳しくは、次のページをご覧ください。  
<https://jpcoar.repo.nii.ac.jp/page/41>
