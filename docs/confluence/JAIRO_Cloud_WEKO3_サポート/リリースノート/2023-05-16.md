# 2023-05-16

_Source: <https://nii-auth.atlassian.net/spaces/JAIROCloudWEKO3/pages/43549422/2023-05-16>_

_階層: JAIRO Cloud（WEKO3）サポート / リリースノート_

## リリースノート

### 主な修正

- インポートにおいて.id（ID）と.uri（URI） の組み合わせが不正な場合のエラーメッセージが出力されずに「サーバ内部エラー」となる問題を修正しました
- Crossref APIのタイトルに改行が含まれている場合にメタデータ自動入力がエラーとなる問題を修正しました
- PDFカバーページを付与した場合に異なるファイル名でダウンロードされる問題を修正しました
- POS\_INDEXのみの一括登録において、同名のインデックスが複数存在した場合、 複数のインデックスに対して更新されずに、若いIndexIDの所属となる問題に対応しました

### 主な変更

- POS\_INDEX　の指定方法が変更となりました。「Index A///Index A-1」のように「///」で階層構造を区切ります。
