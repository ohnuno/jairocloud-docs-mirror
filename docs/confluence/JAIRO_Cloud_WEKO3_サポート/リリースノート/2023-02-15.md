# リリースノート
## v0.9.21
### 主な修正点：
- 一括登録機能での「Change Identifier Mode」が機能していない問題を修正しました。
- SWORDv3経由でのファイルアップロード時にオブジェクトURLが生成されない問題を修正しました。
- 公開日が未来、また閲覧制限のあるインデックスがインデックスリストに表示される問題を修正しました。
- DOI取り下げ後、強制終了または却下時に「Identifier Registration」「Identifier Registration Type」に情報が残る問題を修正しました。
- ファイルを「オープンアクセス日を指定する」で未来日として登録した後に、「オープンアクセス」に変更しても公開日が変わらない問題を修正しました。
- 「著者DBからの入力」で著者データを登録した際に、検索画面に戻らず、著者DBからの入力ダイアログ自体が閉じられてしまう問題を修正しました。
- jpcoar\_1.0によるOAI-PMH出力にてjpcoar:affiliationNameの繰り返しが、jpcoar:affiliationレベルで行われる問題を修正ました。
- ranking処理が従来に比べてかなり遅くなっている問題を修正しました。
- インデックス数が多く、詳細検索機能でインデックス機能をオンにしている場合に、画面表示に時間がかかる問題に対応しました。

### 主な変更点：
- 個別編集中のアイテムに対してのインポートがエラーメッセージも表示されずに完了となる課題に対応しました。
- 改行コードはエキスポート時にbrタグに置換されるが、インポート時に改行に変換されない課題に対応しました。
- OAI-PMHのdatestampがGetRecordとListRecordsで異なる場合がある問題に対応しました。

### 主な追加点：
- JPCOARスキーマ2.0で追加された資源タイプに対応しました。
